Создание локального репозитория Git
	Перейти в папку с проектом 
		cd "D:\mAi test"

	Инициализация Git (один раз) 
		git init 
	
Статусы файлов в Git
	1. Неотслеживаемый (англ. untracked).
	2. Добавленный в индекс, индексируемый (англ. staged, «выдвинутый на плацдарм»).
	3. Изменённый (англ. modified).
	4. Сохранённый, на языке разработчиков это «коммит» или «закоммиченный» (англ. committed, «брошенный в бой»).
	
Как узнать статус файла
	git status
	
Добавить все файлы в индекс
	git add --all
	git add . 

Как сохранить изменения
	git commit -m "My first commit"

Чтобы загрузить изменения на удалённый репозиторий
	git push -u origin master
